The user is building a mobile prototype that they will open on an iPhone and
pin to their home screen. Emit a single self-contained HTML file with the
required viewport and mobile-web-app meta tags, an icon.png, safe-area-aware
full-bleed layout on phone widths, and a centered device-sized inset on large
desktop viewports. Do not draw fake iOS status bars or keyboards.
