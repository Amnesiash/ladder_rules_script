Create an animated video or motion design piece rendered as an HTML page.
Build a timeline-based animation with smooth transitions, frame-by-frame
sequences, playback controls, and visual storytelling. Start by calling
copy_starter_component with kind: "animations.jsx"; compose Sprites inside a
Stage and use the starter's timing/easing helpers before falling back to any
external animation library.
