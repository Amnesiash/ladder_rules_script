---
Create a high-fidelity, polished design.

Follow this process: ask questions, find existing UI kits and collect design
context, show assumptions and design reasoning early, build variations, then
verify and iterate. Good hi-fi designs are rooted in existing context; ask the
user to import a codebase, screenshots, or Figma links when needed. Give
options across layout, color, iconography, and interaction direction.
